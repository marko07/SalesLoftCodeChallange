== README

## Getting Started

### Running natively

1.) Make sure your ruby environment is at least 2.4.6
```
ruby --version
```
2.) bundle install
```
gem install bundle
bundle install
```
3.) Copy the .env.sample to .env and fill out the values
```
cp .env.sample .env
```
4.) Start the development server
```
bundle exec foreman start
```
5.) Navigate to localhost:3000
